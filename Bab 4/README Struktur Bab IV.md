# Struktur Bab IV — Hasil Penelitian

Folder **Bab 4** di repositori ini mengikuti susunan umum **hasil penelitian** untuk skripsi bidang rekayasa perangkat lunak berbasis Scrum. Buka subfolder **secara berurutan** (4.1 → 4.8); nama folder mencerminkan judul subbab yang disarankan pada naskah akhir.

```
Bab 4/
├── README Struktur Bab IV.md          ← berkas ini
├── 4.1 Pendahuluan Bab IV/
│   └── Pendahuluan Bab IV.md
├── 4.2 Hasil Sprint Backlog/
│   └── Hasil Sprint Backlog.md
├── 4.3 Hasil Daily Scrum/
│   └── Hasil Daily Scrum.md
├── 4.4 Hasil Increment Development dan Testing/
│   └── Hasil Increment Development & Testing.md
├── 4.5 Hasil Sprint Review/
│   └── Hasil Sprint Review.md
├── 4.6 Hasil Sprint Retrospective/
│   └── Hasil Sprint Retrospective.md
├── 4.7 Hasil Shippable Product/
│   └── Hasil Shippable Product.md
└── 4.8 Hasil Evaluasi Penelitian/
    ├── Hasil Evaluasi Penelitian.md
    └── Evaluasi SUS/
        ├── README.md
        └── responden/
            └── R-01.md … R-10.md
```

**Catatan:** Jika pedoman kampus memisahkan **hasil** dan **pembahasan** ke bab berbeda, gunakan isi folder 4.2–4.8 sebagai bahan Bab IV, lalu tulis pembahasan pada bab berikutnya dengan merujuk tabel-tabel di sini.
