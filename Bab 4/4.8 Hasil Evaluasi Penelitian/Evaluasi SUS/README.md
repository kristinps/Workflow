# Lampiran evaluasi SUS

Folder ini berisi **satu berkas per responden** untuk kuesioner *System Usability Scale* pada penelitian sistem KPP Biara Loresa SCJ SP3.

- **Format jawaban:** skala 1 hingga 5 untuk setiap pernyataan, sesuai versi Indonesia yang dipakai pada landasan metode.
- **Perhitungan skor:** untuk butir ganjil, kontribusi = jawaban − 1; untuk butir genap, kontribusi = 5 − jawaban. Jumlah kontribusi dikalikan 2,5 sehingga diperoleh skor SUS antara 0 dan 100.
- **Agregat** seluruh responden dirangkum pada `Bab 4/4.8 Hasil Evaluasi Penelitian/Hasil Evaluasi Penelitian.md`.

Nama berkas: `R-01.md` hingga `R-10.md` mengacu pada kode responden pada tabel demografi hasil penelitian.
