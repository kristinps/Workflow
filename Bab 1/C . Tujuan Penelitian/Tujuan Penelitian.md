# Tujuan Penelitian

Berdasarkan rumusan masalah dan implementasi sistem pada folder `Program`, tujuan penelitian ini adalah:

1. Menganalisis dan mendefinisikan kebutuhan pengguna (peserta dan administrator) pada proses pendaftaran serta manajemen Kursus Persiapan Pernikahan (KPP) di Biara Loresa SCJ sebagai dasar pengembangan sistem berbasis web.
2. Merancang dan mengimplementasikan sistem pendaftaran serta manajemen KPP berbasis web yang terintegrasi, meliputi modul pendaftaran peserta, unggah dan verifikasi dokumen, pembayaran QRIS terintegrasi Midtrans, penjadwalan periode, pengelolaan materi, kehadiran, biaya tambahan, dan sertifikat.
3. Mengevaluasi hasil implementasi sistem melalui pengujian fungsional alur end-to-end dan pengukuran usability menggunakan metode System Usability Scale (SUS) untuk menilai tingkat kemudahan penggunaan dan kelayakan sistem.
